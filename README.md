# InfoSapienza_thesis

## Usage:
- Clone the repo
- Run "./deploh.sh"
- It will start a docker container. Go to "http://0.0.0.0:8000/docs" and hit the "search" endpoint for the response

## Dependencies
- Docker
- Faiss DB
- Ollama (not needed during inference)
